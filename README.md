# i2i Academy CI/CD Pipelines Homework

This project demonstrates a simple Continuous Integration and Continuous Deployment pipeline using GitHub Actions.

## Project Content

The project includes:

- A simple Python function
- One unit test using pytest
- One Selenium UI test running in headless mode
- A GitHub Actions workflow file
- A simulated deployment step

## Technologies Used

- Python 3.12
- pytest
- Selenium
- GitHub Actions
- Google Chrome Headless
